1.b) Pela análise dos dados podemos concluir que quanto maior o intervalo entre as medições maior o erro.
Este erro pode ser atingido após termos uma garantia que os valores que estamos a utilizar uns valores corretos.
Esta garantia pode ser dada através do critério do quociente de convergência - este  critério implica que façamos 3 estudos do 
problema, 1 deles usando um passo de integração que vamos chamar de h (identificando este estudo como s), outro dos estudos vai usar h/2 
(identificando como s') e o terceiro vai usar h/4 (identificando como h'').
Deste modo, podemos calcular o quociente de convergência através do cálculo de (s' - s)/(s'' - s') e podemos provar a validade dos nossos valores
se este valor for aproximadamente 2^(ordem do método). Como neste caso estamos a utilziar o método de simpsons, método de quarta ordem, para podermos
identificar os erros teriamos de obter um quociente de convergência de aproximadamente 16.
Pela análise dos dados podemos concluir que este valor é alcançado e finalmente podemos calcular o erro absoluto. Este arro vai ser dado por (s'' - s')/x, 
sendo este x o valor de (2^ordem do método) - 1 (neste caso 15).
Assim, fizemos duas análises aos valores dados:
	. Primeiro usamos um valor inicial de h = 8 para o passo de integração (usando posteriormente h = 4 e h = 2 para os restantes estudo) e obetivemos o valor:
		.-2.2717684055733116
	. Posteriormente vamos usal o valor inicial de h = 4 para o passo de integração (usando de seguida os valores h = 2 e h = 1) e obtivemnto um valor significativamente
menor:
		.-0.14198539985785222
(ambos as tentivas obtiveram um valor de qc de aproximadamente 16)
E, assim, podemos concluir que quanto menor for o intervalo entre as medições, menor o erro que podemos obter os nossos cálculos. Porém, como teremos um intervalo menor entre
as medições iremos ter mais iterações o que não torna o método tão eficiente.
